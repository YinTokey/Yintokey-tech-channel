 # Skill Name
Infographic Style-1

1. **Learn the style.** Analyze references for layout, typography, palette, density, hierarchy, icon style, and brand cues.
2. **Lock the core drivers:**
   - Purpose → aspect ratio + density
   - Archetype → layout type
   - Content → exact text/data
   - Audience/tone → visual register
   - Quantity → variants/count
   - Brand → consistency rules
3. **Generate via GPT Image 2** applying style, archetype, content, and brand constraints.
4. **Review every output** for factual accuracy, phantom content, archetype fidelity, layout sanity, and brand compliance. Web-search if facts are uncertain or current.
5. **Auto-fix and retry** with targeted corrections (max 5). If still failing, report what failed and what needs clarification.
